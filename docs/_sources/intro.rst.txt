Getting started
===============

*Prayas* is currently only available at https://github.com/Avira/prayas.
Follow the instructions explained in the ``README.md`` to install it.

Use *prayas*:

>>> from prayas import *

Learn *prayas*:

* Read `Bayesian A/B Testing for Business Decisions <https://arxiv.org/abs/2003.02769>`_ by Shafi Kamalbasha and Manuel J. A. Eugster (2020) for an explanation of the Bayesian methodology implemented in this package
* Check out the :ref:`ref-examples` to see *prayas* in action
